# agent-playground-static

Tiny Vite + React app used to test Open-Inspect's **Publish** feature
(`publish-app` tool → Cloudflare Pages deploy).

The app has nothing interesting in it — the point is just for the agent
to modify a file (e.g. change the heading), click **Publish**, and
get back a permanent `*.pages.dev` URL.
